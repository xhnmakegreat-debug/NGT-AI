## 🔄 进度跟踪流程

1. **阶段1: 独立观点生成** (20-30%)
   - 消息: "正在生成独立观点..."
   - 消息: "已完成独立观点生成，收集到X个观点"

2. **阶段3: 交叉评分** (40-50%)
   - 消息: "正在进行交叉评分..."
   - 消息: "已完成交叉评分，收集到X份评分表"

3. **阶段4: 分数聚合** (60-70%)
   - 消息: "正在聚合评分..."
   - 消息: "已完成分数聚合"

4. **阶段5: 修正/捍卫** (80-85%)
   - 消息: "正在进行观点修正或捍卫..."
   - 消息: "已完成观点修正，收集到X个最终观点"

5. **阶段6: 裁判汇总** (90-95%)
   - 消息: "正在进行裁判汇总分析..."
   - 消息: "已完成裁判汇总"

6. **完成** (100%)
   - 消息: "正在生成最终报告..."

## 📋 报告内容结构

### 执行摘要
- 处理时长
- 参与AI数量
- 初始观点数
- 观点修正数
- 观点捍卫数
- 平均得分
- 完成率
- 错误次数

### 各讨论员初始观点
- AI ID和模型名称
- 详细观点内容

### 交叉评分详情
- 评分者信息
- 被评分AI的得分和理由

### 裁判综合结论
- 合并观点
- 亮点观点
- 风险评估
- 最终建议

### 行动建议
- 最终决策详情
- 修正/捍卫情况
- 得分统计
